# 진홍건 Portfolio

주니어 백엔드 개발자 진홍건의 포트폴리오 사이트입니다.

🔗 https://hh-gun.github.io

## 기술 스택

- HTML5 / CSS3 (Vanilla)
- Pretendard 폰트
- GitHub Pages

## 구조

```
.
├── index.html              # 메인 페이지
├── assets/
│   ├── css/
│   │   └── style.css       # 스타일시트
│   └── js/                 # (필요 시 스크립트)
└── README.md
```

## 로컬 실행

별도 빌드 없이 `index.html`을 브라우저에서 바로 열거나, 간단한 정적 서버로 실행할 수 있습니다.

```bash
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```
